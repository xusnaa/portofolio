Network Scan
Build a network scanner that discovers active devices on a given network. You can use tools like Nmap or develop a simple scanner that pings devices to identify live hosts and gathers information such as open ports.
Steps to follow
1. Network Discovery: Initiate network discovery by either utilizing a network scanning tool like Nmap or implementing a custom solution. This involves sending requests to IP addresses within the specified network range.

2. Host Identification:Identify active devices by analyzing responses to network requests. A simple ping sweep can be used to determine live hosts by checking for responses to ICMP (Internet Control Message Protocol) echo requests.

3.Port Scanning:Perform port scanning on live hosts to identify open ports and services. This step involves sending connection requests to different ports and analyzing the responses to determine which ports are actively accepting connections.

4. Information Gathering:Gather additional information about the discovered devices, such as device type, operating system, and services running on open ports. This information can be obtained by analyzing the responses received during the network and port scanning processes.
